About this project

In general

Opensource technologies

This site relies heavily on the following opensource technologies:

  • it is written in Python programming language
  • it uses Django web framework
  • it molests South for Django's database schema migrations
  • it is performed in your browser by Bootstrap and by that, coincidentally, JQuery
  • it is hosted on Debian linux distro running Nginx HTTP server.
  • it is developed on Fedora linux distro and often edited by VIM editor.
